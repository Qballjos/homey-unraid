## ğŸ¤ Discussion Guidelines

### Do's âœ…
- **Search first** - Your question might be answered already
- **Be descriptive** - Provide context and examples
- **Share logs** - Use code blocks for error messages
- **Mark solved** - Mark answers that solve your problem
- **Be respectful** - We're all here to help each other
- **Share knowledge** - Answer questions you know
- **Vote** - Upvote helpful posts and comments

### Don'ts âŒ
- **Don't spam** - One post per topic
- **Don't hijack** - Create new discussions for new topics
- **Don't demand** - Be patient, volunteers help when they can
- **Don't share secrets** - Never post API keys or passwords
- **Don't cross-post** - Pick either Discussion OR Issue, not both
