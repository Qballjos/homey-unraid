	• Memory efficient - Only ~40MB RAM usage

Quick Setup

1. Enable GraphQL API in Unraid (Settings → Management Access)
2. Generate API Key (Settings → API Keys)
3. Add device in Homey and configure URL & API key
4. Start automating!
